# Demo script (6 minutes)

**Setup:** `npm start`, full app in one window, pop-up (`/?popup=1`) beside a Cursor window.

1. **Hook (30s).** "Sarah runs invoice automation in Finance. When she leaves, her prompts,
   workflows, and the reasons they worked usually leave with her. Here they don't."

2. **Pop-up recall (90s).** As Maya (employee), in the pop-up type *"expense automation"*.
   The context pack surfaces Sarah's **Invoice OCR workflow** with *why it worked* (300 DPI
   preprocessing), grounded expense policy claims with tier badges. Click → provenance drawer
   walks claim → Slack thread → GitHub README. Copy the pack into Cursor: "this is exactly what
   the Engrammic MCP returns to the agent at task start."

3. **Governance (60s).** Still as Maya, recall *"board reporting pack"* → nothing, with
   "1 hidden by role scope". Re-login as Priya (executive) → the restricted capability appears.
   Show Scopes matrix: roles come from WorkOS Directory Sync.

4. **Adjudication (90s).** Ask *"annual leave days"* → answer is **contested**: Drive policy says
   15, Slack rumor says 20. Open Conflicts, keep the authoritative claim. Recall again → rumor is
   superseded and gone; Knowledge now shows a wisdom-layer adopted belief with full chain.

5. **Org intelligence (60s).** Analytics: hottest capabilities (what's actually reused),
   knowledge gaps ("SOC2 evidence checklist" — asked, unanswered), duplication (Finance and Ops
   built the same receipt checker independently).

6. **Close (30s).** Install MCP page: one snippet connects Cursor/Claude Code/Codex.
   *"Employees keep their tools. The org keeps the capability. Competitors store memories —
   we adjudicate claims."*
